用sublime text套sb-admin的過程

2015-06-29 869 words 2 mins read times read

Contents

今天想要套套看不同的Thene 找了sb-admin-2來用 參考這篇教學

###有幾個重點 先按照前兩個步驟完成,最後一個加THEME的動作比較複雜 ihower的教學 1. 安裝gem:Gemfile裡面加上gem 'bootstrap-sass'和 gem 'autoprefixer-rails' 2. 在assets/javascripts/application.js下加入

//= require bootstrap-sprockets

###加入樣式

  1. 把抓下來的THEME檔，CSS、JS都放進assets/stylesheets、assets/jacascripts的資料夾，包括plugins/
  2. 如果THEME檔案中有font檔，就要直接在assets/下建立fonts資料夾
  3. 如果有font-awesome，要另外把font-awesome/css下的*.css檔一併拿到assets/stylesheets下
  4. 有image檔就統一方在assets/images下
  5. 把下載的檔案中index.html的內容複製到您預設的首頁，記得刪除預設的css、js路徑，雖然可能都還看得正常，但還是就報錯了 先從develope tool看source是否有成功讀取

  6. FontAwesome’s css 檔案需要被轉成erb檔，所以用Rails path helper 來修復bug

    1
2

    src: url('<%= asset_path("fontawesome-webfont.eot") %>?v=4.1.0');
src: url('<%= asset_path("fontawesome-webfont.eot") %>?#iefix&v=4.1.0') format('embedded-opentype'), url('<%= asset_path("fontawesome-webfont.woff") %>?v=4.1.0') format('woff'), url('<%= asset_path("fontawesome-webfont.ttf") %>?v=4.1.0') format('truetype'), url('<%= asset_path("fontawesome-webfont.svg") %>?v=4.1.0#fontawesomeregular') format('svg');

  7. 接著要修復js的bug,我們stub兩個在瀏覽器開發者工具中出現error的js檔案，這兩個檔案是THEME在demo時一起下載，我們用stub keyword把它關閉，進到application.js

    1
2

    //= stub plugins/flot/flot-data
//= stub plugins/dataTables/dataTables.bootstrap

stubbing會告訴Rails忽略這些檔案，但我們先不刪除檔案留著備用
