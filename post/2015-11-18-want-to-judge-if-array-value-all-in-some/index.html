want to judge if array value all in some?

2015-11-18 483 words 1 min read times read

Contents

遇到的問題是如果沒有搜尋條件，就出現A，有搜尋但沒符合條件出現B，有搜尋有條件出現C 因此在controller要 ####1.判斷是否出現搜尋的session: 如果全部判斷每個是否為空會很麻煩

1

params[:keywords] == '' && !session[:search]

再來要判斷是出現空值或nil ####2.controller的method不能直接用在view裡面: 所以要在controller裡面增加helper_method ####3.debug的方式 - 在controller 的object.inspect - 在view中呼叫方法看值 找到使用方法all? ###controller的測試

helper_method :first_come?


def first_come?
	#fordebug  session[:search].inspect
  #猜測是否為空 session[:search].values  => 在view出現{nil,nil,nil,nil} 
  #最終版正確方法
  params[:keywords] && session[:search].values.all?{|val| val==nil} 
end
