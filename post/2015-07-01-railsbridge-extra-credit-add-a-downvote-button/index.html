RailsBridge Extra Credit : Add a downvote button

2015-07-01 375 words 1 min read times read

Contents

已經作到最後一個步驟，現在在做加分項目： 在文章投票可以減分 (http://installfest.railsbridge.org/intro-to-rails/credits_and_next_steps) 結果我在 controllers/topics_controller.rb

def downvote
  @topic = Topic.find(params[:id])
  @topic.votes.destroy 
  redirect_to(topics_path)
end

結果說沒有downvote這個方法 發現是route.rb沒有在member下面加入路徑，於是加入

1
2
3
4

config/routes.rb
resources :topics do
  member do
    post 'upvote'
    post 'downvote'
  end
end

還是有錯，於是找問題發現這個網站的解法，發現因為倒出來的是array， 沒有指定哪一個被刪，因為我們要刪的是最後一個，所以要加入last的方法 找到這個(http://discourse.codenewbie.org/t/railsbridge-curriculum-questions/594)

1
2
3
4
5

def downvote
  @topic = Topic.find(params[:id])
  @topic.votes.last.destroy # 這裡加入last
  redirect_to(topics_path)
end

  • 在 view 的部分，仿造 +1 button 新增 -1 button，但 upvote 改成 downvote
