Start using phantomjs

2015-07-15 180 words 1 min read times read

Contents

###安裝 官網的步驟： 1. 下載，然後解壓縮。 2. 在terminal中執行，格式為{解壓縮目錄}/bin/phantomjs {欲執行的腳本路徑} 後來發現在Mac OS X上有更快更簡單的方式，就是利用Homebrew!! 1. brew update && brew install phantomjs，就安裝好了！ 2. 檢查安裝的版本 phantomjs --version 3. 在想要執行的專案路徑下執行phantomjs XXXXX.js

這篇教學有詳細步驟，Testing JavaScript with PhantomJS
